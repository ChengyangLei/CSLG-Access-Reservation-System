如果您需要使用本软件来管理相关的门禁系统，则建议您先阅读以下几点内容：

========================================================================================
1.本软件需要连接相关数据库，相关数据库资源可以通过添加数据源来进行（如果您没有事先创建数据源）
	如果您不懂得如何建立数据源，则按照以下步骤来执行：
	1）在系统管理工具中找到数据源（JDBC）并打开。
	2）如果您定义本软件使用电脑中所有用户都能够使用此数据源，则点击横向菜单“系统DSN”，如果不需要，则在原菜单“用户DSN”中操作即可。
	3）点击界面右方添加，选择SQL Server（在此仅使用SQL Server做介绍）。
	4）对您的数据源命名，注意（如果自定义数据源，请使用CSLG Access reservation system作为数据源名称）。
	5）选择您要操作的数据库服务器，点击下一步。
	6）选择您默认要操作的主要数据库。
	7）完成
